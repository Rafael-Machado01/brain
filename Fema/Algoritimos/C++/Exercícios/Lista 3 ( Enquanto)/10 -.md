# Entrada
Valor inicial do Numerador (Exemplo 5);
Valor inicial do Denominador (Exemplo 1)
Valor zerado para Somatoria (Para lixo de memória)
# Processamento 
S + N1/N2 apos calcular aumente 3 do Numerador e 1 do Denominador

# Saída 
Valor final.

```
ALGORITIMO DEZ
VAR
N1,N2,S : REAL;
INICIO
N1 <- 5
N2 <- 1
S <- 0
ENQUANTO(S < 32)
FAÇA 
	S <- S + N1 / N2;
	N1 <- N1 + 3;
	N2 <- N2 +1;
IMPRIMA "A soma é ", S;
FIM
````
