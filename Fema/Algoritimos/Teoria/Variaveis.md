
# Vetor = Array.
Um dimensão.

```
v[5];

0  1  2  3  4  

Cabe 5 variaveis

Sempre do mesmo tipo.

Sempre começa com 0 igual arrays.
````

Indiçe

```
V[2] = 5

0 1 2 3 4
0 0 5 0 0
Imprima V[2];

````

Leia um vetor de 50 até mesmo para preencher.

```
ENQUANTO(I <= 50) 
	FAÇA
	Imprima V[I];
	I <- I + 1;
	
FIM;
````
